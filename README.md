Реализована система UI, анимаций к ним на основе корутин и система сценариев на оснвое Task, примененных в безопасном контексте. Инструменты не самые удобные, но по условиям тестового задания запрещено использование сторонних библиотек, кроме родных от Unity.
Также ради интереса поэкспериментировал с шейдерами, чтобы получить скругленную маску и эффект градиента.
В качестве аниматоров с настройками решил применить обычные классы, как наиболее удобные, однака из-за запрета на использование удобных утилит для инспектора, реализовал самостоятельно несколько кастомных компонентов для редактора и воспользовался [SerializeReference]. Все работает, но, к сожалению, прокинутые и натсроенные классы по прежнему может слетать при внесении изменений в коде.
